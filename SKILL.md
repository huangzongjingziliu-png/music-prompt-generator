# 音乐复刻提示词生成器

音乐复刻提示词生成器是一款专业的AI音乐分析工具，能够根据用户提供的音乐链接或上传的音频文件，自动识别音乐特征并生成适用于各大AI音乐平台的高质量复刻提示词。

## 核心功能

### 一、音乐分析能力

本技能支持多种音乐输入方式，能够全面分析音乐的各项特征参数。

**支持的音乐输入格式：**

- **音乐链接分析**：支持网易云音乐、QQ音乐、Spotify、Apple Music、YouTube Music等主流音乐平台的链接分析
- **本地音频文件**：支持 mp3、wav、aac、m4a、flac、ogg 等常见音频格式
- **文件大小限制**：建议单个文件不超过 50MB 以确保分析准确性

### 二、音乐特征识别

系统会自动识别并输出以下音乐特征，为后续提示词生成提供精准依据。

**风格流派识别：**
古风、流行、电子、摇滚、古典、民谣、爵士、说唱、嘻哈、乡村、拉丁、RNB、电影配乐、游戏音乐等。

**情绪基调分析：**

| 能量等级 | 情绪关键词 |
|---------|-----------|
| 高能量 | 激昂、热血、战斗、爆发、欢快、兴奋 |
| 中能量 | 深情、温柔、浪漫、史诗、壮观 |
| 低能量 | 安静、平静、忧伤、孤独、空灵 |

**速度 BPM 识别：**

| 速度分类 | BPM 范围 | 描述 |
|---------|---------|------|
| 慢速 | 60-80 | 舒缓、深沉 |
| 中速 | 80-110 | 平稳、流畅 |
| 快节奏 | 110-140 | 活力、动感 |
| 极速 | 140+ | 激烈、狂热 |

**调式识别：**
大调、小调、五声音阶、七声音阶、民族调式、特殊调式等。

**乐器识别：**
根据音乐内容识别主要乐器和次要乐器组合。

```

[instruments], [texture]
```

## 使用方法

### 第一步：提供音乐素材

用户可以通过以下方式提供需要分析的音乐：

1. **粘贴音乐链接**：提供网易云、QQ音乐、Spotify等平台的歌曲链接
2. **上传音频文件**：直接上传本地音乐文件（支持 mp3、wav、aac、m4a 等格式）
3. **描述音乐特征**：如果无法提供音频，可描述音乐的大致特征

### 第二步：等待音乐分析

系统将自动进行以下分析步骤：

- 音频波形特征提取
- 频谱分析与音色识别
- 节奏模式检测
- 情绪特征分析
- 乐器成分识别

### 第三步：获取分析结果

分析完成后，系统将输出完整的分析报告，包括特征分析、提示词和曲名建议。

## 注意事项

1. **链接有效性**：请确保提供的音乐链接可正常访问，部分平台可能存在地域限制
2. **音频质量**：上传的音频文件质量越高，分析结果越准确
3. **版权提示**：本工具仅用于音乐分析和创作灵感获取，请勿用于侵犯版权的用途
4. **结果调整**：AI音乐生成具有一定随机性，可根据实际结果微调提示词
5. **多语言提示词**：建议优先尝试英文提示词，国际平台对英文理解更准确

## 应用场景

- **音乐创作灵感**：为音乐人提供创作方向和提示词参考
- **风格模仿练习**：学习特定音乐风格的编曲和制作特点
- **AI音乐复刻**：使用AI工具复刻喜欢的音乐风格
- **教学演示**：用于音乐理论教学中的风格分析演示
- **版权研究**：分析音乐作品的风格特征和制作要素
